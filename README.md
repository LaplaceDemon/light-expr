# light-expr
解析表达式抽象语法树

解析表达式

```java
// 表达式
String expression = "x = a + b + c*d + sin(e) + max(e,f)";

// 构建解析器
Parser parser = new Parser(expression);

// 开始解析，得到结果
BaseExpression be = parser.parse();

// 打印
System.out.println(be);

```

打印结果：
```
 = [ x , + < + < + < + < a , b > , * < c , d > > , sin < e > > , max < [e, f] > > ]
```

这是一个抽象语法树的打印结果，它表示表达式是这样一个树形结构：
```
         =
 /               \
x                 +
           /             \
         +               max()
      /     \                |
     +     sin()            e,f
   /   \       |
  +     *      e
 / \   / \
a   b c   d
```
